# BioGreen Website

BioGreen Fertilizer & Gas Manufacturing PLC is a leading agricultural solutions company in Ethiopia, specializing in regenerative organic farming and sustainable plant-based fertilizers.

## Features
- **Regenerative Agriculture**: Products made from upcycled avocado and coffee by-products.
- **Sustainability Driven**: Circular economy model transforming waste to wealth.
- **Empowerment**: Supporting local farmers and a majority-women workforce.

## Tech Stack
- [Next.js](https://nextjs.org/) (App Router)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Lucide React](https://lucide.dev/) (Icons)

## Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to see the result.

### Build
To build for production:
```bash
npm run build
```

## Deployment
This project is configured for easy deployment on [Vercel](https://vercel.com/). Connect your GitHub repository to Vercel and it will automatically detect the Next.js settings.
